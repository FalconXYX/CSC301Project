# Definition of Done

A story is completed when all of the following conditions have been satisfied:

1. All the required acceptance criteria defined in the story are completely implemented.

2. The new or modified functionality works as expected and does not negatively impact other existing functionalities.

3. Newly-written code follows coding standards that were established for this project.

4. Each member of the team has reviewed and accepted the final changes to the code.
   - Reviews will happen using formal pull requests on GitHub.
   - More detailed reviews can also take place via a meeting on our chosen communication platform.

5. The final, reviewed code has been merged into the main branch.
