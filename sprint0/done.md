A story is completed when all of the following conditions have been satisfied:
1. All the required acceptance criteria defined in the story are completely implemented.
2. The new functionality works as expected and does not negatively impact existing functionalities.
3. The code has followed the coding standards previously established for this project.
4. Each member of the team has reviewed and accepted the final changes to the code (through a review process on Discord and/or through a pull request review process via GitHub).
5. Finally, the last version of the code was merged into the main branch.
