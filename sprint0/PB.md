### User Story Template

> As a \[persona\], I \[want to\], \[so that\].

### Priority Legend

|Priority|Description|
|-:|:-|
|**Critical**|This is a breaking change that _must_ be implemented before all others.|
|**Time&#8209;sensitive**|This is a necessary task item that _must_ be implemented in this sprint.|
|**Regular**|This is a task item that _should_ be implemented in this sprint, but will not cause an issue if it remains incomplete.|
|**Cosmetic**|This is a task item that should only be implemented if time permits and all other higher-priority changes have been completed.|

*Tasks should be completed in order of priority.*
